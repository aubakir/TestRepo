# Замерятель скорости интернета

Скрипт последовательно скачивает файл по указанному URL несколько раз, замеряет время каждого запроса и выводит скорость соединения в МБ/с.

## Что делает

1. Принимает адрес тяжёлого файла (картинка, архив и т.п.)
2. Выполняет **10 последовательных HTTP-запросов** к этому адресу
3. Дожидается полного получения ответа на каждый запрос
4. Вычисляет среднее время запроса и объём скачанных данных
5. Выводит в консоль скорость в МБ/с — среднюю, медианную, мин., макс. и стандартное отклонение

## Настройка

Все параметры задаются в файле `speedtest/config.py`:

```python
# Адрес файла для скачивания (лучше тяжёлый файл от 10 МБ)
URL: str = "https://..."

# Сколько раз последовательно скачивать файл
NUM_REQUESTS: int = 10

# Таймаут одного запроса в секундах
TIMEOUT_SEC: int = 30
```

## Запуск

```bash
python app.py
```

## Структура проекта

```
app.py                # точка входа CLI
speedtest/
├── __init__.py       # публичный API пакета
├── config.py         # настройки по умолчанию
├── models.py         # модели данных (RequestResult, SpeedReport)
├── client.py         # HTTP-загрузка
├── measurer.py       # оркестрация N запросов
└── reporter.py       # вывод в консоль
```

## Требования

- Python 3.11+
- Только стандартная библиотека, сторонние зависимости не нужны
